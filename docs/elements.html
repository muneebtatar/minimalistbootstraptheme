Panel card

Panel cards are cards with a header image, a header, a small description and a long description. These are used for showing blog posts (as you saw on the home/landing page) but you can use them for many other various purposes as well.

<div class="panel panel-default panel-card">
   <div class="panel-heading">
      <img src="headerimglink.jpg">
   </div>
                    
   <div class="panel-body text-center">
      <h4 class="panel-header"><a href="">This is a header</a></h4>
      <small>A short description goes here.</small>
   </div>
    
   <div class="panel-info">
       <p>A Long description goes here.</p>
   </div>
</div>

Changing background image for headers

You can also change the background image for headers by editing the main minimalist css file or using inline css styles.

Changing huge header background image

The huge header background image can be change from the main minimalist css file or you can change it using inline css styles (used below).

It has to be noted that the huge header overlays the background image with another image (overlay image), so in inline css styles the overlay image has to be linked too.

<header class="huge-header" style="background-image: url('../img/overlay.png'), url('/link/to/background.jpg');">
 <div class="container">
  <div class="row">
   <div class="col-lg-8 col-lg-offset-2 col-md-10 col-md-offset-1">
    <div class="content">
     <h1 class="center mb">Minimalist</h1>
     <h2 class="subheading center">A fine theme twitter bootstrap theme</h2>
      <div class="text-center">
       <div class="btn btn-outline">Check the features</div>
      </div>
     </div>
    </div>
   </div>
</header>
