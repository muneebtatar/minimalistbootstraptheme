Minimalist

Goodies

Minimalist is built using the twitter bootstrap framework version 3.3.2. The Minimalist theme also comes with these

  • Icon packs
  • Kameleon
  • krispicons
  • Logotypes
  • Font Awesome
  • Fonts
  • Source Sans Pro 200, 300, 400, 600, 700, 900
  • Libraries
  • jQuery

Minimalist requires jQuery in order to function properly (Twitter Bootstrap also requires it). The icon packs are optional, you can also choose more "free" icon packs that suits your taste from Iconfinder. Before you use icons from Iconfinder in your projects make sure you read the article on licenses.
